# Support

## Where to go

- Questions about installing or using Orbit: open a GitHub issue
- Feature ideas: open a GitHub issue with the feature template
- Security problems: follow [SECURITY.md](SECURITY.md)
- General project updates and launch links: [orbitcodex.org](https://orbitcodex.org)

## Good issue reports include

- your Orbit version or commit
- macOS version
- whether you installed via PKG or DMG
- whether the issue is about permissions, Codex auth, cloud voice, or overlay behavior
- screenshots when relevant

## What Orbit does not provide

- a hosted backend
- account management outside the user’s own ChatGPT / Codex / OpenAI setup
- bundled analytics in the open-source build
